# BurgEDUI - Your new desktop environment
BurgEDUI is made to be a simple desktop environment for **linux** based on the **mir** display server by canonical, and **LayerShellQt** by the kde team.
This project was made to give a simple, clean, MacOS like experince to linux, with customization and features to improve the experince.
## Apps
* BurgEDUI includes apps like a:
* file manager
* terminal app
* text editor 
* and more to come!\
\
note: BurgEDUI is still a work in progress, so final apps are not guaranteed. apps listed here will be included in the product, with more soon.
### features
BurgEDUI will have a standard set of features, with apps and customization being top priority for development. It will include smooth animations, and simple layout presets for what you need.
